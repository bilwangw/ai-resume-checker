# ai-resume-checker
AI powered resume checker, compare to job descriptions and see if your resume matches


Installation:

``` pip install -r requirements.txt```

Follow these instructions to install spaCy: https://spacy.io/usage:

Should be: (this uses ```en_core_web_lg``` not ```en_core_web_sm```)
```
pip install -U pip setuptools wheel
pip install -U spacy
python -m spacy download en_core_web_lg
```
